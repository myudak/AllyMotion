# Install Project

Done with the pre-requisites?

Here are the steps to be taken after that :

1. Clone the repository:

```bash
git clone https://github.com/myudak/AllyMotion
```

2. Change directory:

```bash
cd AllyMotion
explorer .
```

3. Open the project in android studio

4. Sync project with grade

5. Run the app
